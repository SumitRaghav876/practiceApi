# REST API Assignments

7 REST APIs built from scratch using Node.js + Express.

## Projects

| Project | Description |
|---|---|
| 📁 first api | basics of REST API |
| 📁 NotesApi | search by title & content, filter by tag, pagination |
| 📁 booksApi | filter by genre, availability, sort by price, stats |
| 📁 LibraryBooksApi | advanced book management with search & pagination |
| 📁 StudentsApi | sort by marks, pagination, stats |
| 📁 ProductApi | filter, price range, sort, patch stock, stats |
| 📁 EcomApi | ecommerce product management |

## Tech Stack
Node.js · Express.js · REST APIs · JSON file storage
